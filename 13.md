 # Usi ka swithch version
  ```Java
   
 
  class Solution {
    public int romanToInt(String s) {
        int ans = 0;
        int prev = 0;   // previous value

        for (int i = s.length() - 1; i >= 0; i--) {
            int val = getValue(s.charAt(i));

            if (val < prev) {
                ans -= val;   // subtractive case (like IV = 5-1)
            } else {
                ans += val;   // normal case
            }

            prev = val;       // update prev
        }

        return ans;
    }

    private int getValue(char c) {
        switch (c) {
            case 'I': return 1;
            case 'V': return 5;
            case 'X': return 10;
            case 'L': return 50;
            case 'C': return 100;
            case 'D': return 500;
            case 'M': return 1000;
        }
        return 0;
    }
}

```

 ### Sabse pehle mujhe Roman numeral rules yaad karne padhe:



* I = 1, V = 5, X = 10, L = 50, C = 100, D = 500, M = 1000

* Mostly left se right add karte hain.

* Exception: Agar chhota symbol bada wale ke left me ho → subtract.

* IV = 4, IX = 9, XL = 40, XC = 90, CD = 400, CM = 900

```java
class Solution {
    public int romanToInt(String s) {
        Map<Character, Integer> map = new HashMap<>();
        map.put('I', 1);
        map.put('V', 5);
        map.put('X', 10);
        map.put('L', 50);
        map.put('C', 100);
        map.put('D', 500);
        map.put('M', 1000);

        int sum = 0;
        for (int i = 0; i < s.length(); i++) {
            int value = map.get(s.charAt(i));
            if (i + 1 < s.length() && value < map.get(s.charAt(i + 1))) {
                sum -= value;
            } else {
                sum += value;
            }
        }
        return sum;
    }
}
```

## **1. Problem recap**

Roman numerals me kuch cases normal addition se nahi bante, unme subtraction hota hai:

* `IV` = 4 (5 - 1)
* `IX` = 9 (10 - 1)
* `XL` = 40 (50 - 10)
* `XC` = 90 (100 - 10)
* `CD` = 400 (500 - 100)
* `CM` = 900 (1000 - 100)

Baaki sab normal addition hota hai.

## **4. Special subtraction case handling**

### Example: 'I'

```java
if (c == 'I') {
```

* Agar `c` = 'I' aur agla character 'V' ya 'X' hai → special value (4 or 9) add karo.
* `i++` → agla character skip karte hain kyunki wo already count ho gaya.
* `continue` → loop ke next iteration me chale jao.
